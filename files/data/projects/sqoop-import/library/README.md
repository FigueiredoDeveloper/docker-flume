# library

Biblioteca com funcionalidades gerais para os projetos B2W/Labs.

Este projeto geralmente é utilizado como parte integrante de outros projetos (sub-modulo git).


### Features

* Storage multiplataforma: salva dados em mais de um storage ao mesmo tempo (Oracle, HDFS, Impala, etc);

* Ferramenta para envio de notificações por email/SMTP;

* Facilitadores para tragamentos de argumentos de linha de comando, geração de logs e tratamento de arquivos de configuração, etc.


### Instalação

```bash
sudo easy_install -U setuptools
sudo pip install -r requirements.txt
```
